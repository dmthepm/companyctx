# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc-2.1], a widely
used open-source code of conduct released under [Creative Commons Attribution
4.0 International][cc-by-4].

The full text is available at the link above. By participating in this project
you agree to follow it.

## Reporting

Instances of abusive, harassing, or otherwise unacceptable behavior may be
reported to the project maintainers by opening a private security advisory on
the repository, or by emailing the address listed in the repository profile.
Reports will be reviewed and investigated promptly and fairly.

All maintainers are obligated to respect the privacy and security of the
reporter of any incident.
